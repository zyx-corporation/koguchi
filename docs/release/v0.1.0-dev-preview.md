# v0.1.0-dev-preview

Koguchi v0.1.0-dev-preview is the first Developer Preview of the Context-Aware Harness reference implementation.

It provides a minimal, reviewable architecture for routing tool execution through an accountable side-effect chokepoint.

## Highlights

- ToolProxy as the single side-effect chokepoint
- PolicyGate for envelope / policy decisions
- RuntimeBoundary for best-effort runtime checks
- ServiceRuntime as an accountable execution surface
- AuditEventSink abstraction
- Reconciliation v2
- Redaction / Secret Safety
- RDE / T-RDE support
- Minimal executable example
- ADR-001 to ADR-021

## Try it

```bash
ruff check .
pytest
mypy src
python examples/minimal_tool_proxy.py
```

## Security and limitations

Koguchi is not a security sandbox.

The current Python implementation provides best-effort runtime constraint checks only. Strong isolation is deferred to Rust chokepoint and OS/container-level enforcement.

See:

- `docs/known-limitations.md`
- `docs/architecture.md`
- `docs/adr/ADR-020-runtime-hardening.md`
- `docs/adr/ADR-021-service-runtime.md`
