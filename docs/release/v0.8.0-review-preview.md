# v0.8.0-review-preview

Koguchi v0.8.0-review-preview is a Review Preview of the Context-Aware Harness reference implementation.

It provides a static, read-only HTML review artifact over audit records, reconciliation jobs, reconciliation results, execution backend counts, and Rust chokepoint availability.

## Highlights

- ToolProxy as the single side-effect chokepoint
- PolicyGate / RuntimeBoundary / ServiceRuntime / ExecutionBackend separation
- Optional Rust Chokepoint backend
- Persistent Audit Store
- Reconciliation Scheduler
- Persistent Reconciliation Result Store
- Dashboard Observation Plane
- Static HTML Dashboard Report
- HTML escaping
- No form / no button / no script
- ADR-001 to ADR-028

## Try it

```bash
ruff check .
pytest
mypy src

python examples/minimal_tool_proxy.py
python examples/persistent_audit_store.py
python examples/reconciliation_scheduler.py
python examples/rust_chokepoint_spike.py
python examples/dashboard_observation.py
python examples/serviceruntime_chokepoint_backend.py
python examples/dashboard_static_html_report.py

cargo test --manifest-path crates/koguchi-chokepoint/Cargo.toml
cargo build --manifest-path crates/koguchi-chokepoint/Cargo.toml
```

## Static report scope

The static HTML report is read-only.

It does not:
- execute tools / approve actions / mutate logs
- rerun jobs / repair or rollback / call chokepoint
- provide web server / remote API / certify compliance / prove security

## Security and limitations

Koguchi is not a security sandbox. See `docs/known-limitations.md`.
